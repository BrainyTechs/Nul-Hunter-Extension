# Change Log

All notable changes to the "nul-hunter" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.0.0] - 2026-03-23

### Added
- Initial release with core functionality
- Find and clean NUL (null byte) files in projects
- Command palette integration for finding and cleaning NUL files
- Context menu integration in file explorer
- Progress indicator during cleaning process
- Information about cleaned file count and size

### Changed
- Added bilingual README (English and Turkish)

## [Unreleased]
- Support for custom file pattern exclusions
- Option to automatically scan on project open
- Statistics dashboard for cleaning history
